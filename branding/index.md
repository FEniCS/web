# FEniCS branding

## Components of FEniCSx and legacy FEniCS
The components of FEniCSx are:

* DOLFINx
* FFCx
* Basix
* UFL

The components of legacy FEniCS are:

* DOLFIN
* FFC
* FIAT
* UFL legacy

These should always be written using the capitalisation above.
